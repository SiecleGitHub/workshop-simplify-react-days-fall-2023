# Working Repo for The ReactDays Berlin 2023 Workshop Simplify React

## Setup

Install dependencies with NPM

```bash
$ npm install
```

## Run

This will start the development server, consult the console for the URL to open in your browser.

```bash
$ npm run dev
```
